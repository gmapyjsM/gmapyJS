O Gmapy.js
Realizací návrhu frameworku je vytvořený Gmapy.js framework vycházející z Google Maps API. Gmapy.js umožňuje pracovat s funkcemi API jednodušeji a přehledněji. Obsahuje navíc také některé rozšiřující funkce. Framework je napsán v jazyce JavaScript s využitím Google Maps API a jQuery knihovny.
